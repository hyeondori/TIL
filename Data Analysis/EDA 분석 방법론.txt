데이터 분석에는 크게 두 가지의 접근 방법이 있다.

첫번째는 확증적 데이터 분석(CDA: Confirmatory Data Analysis)으로 가설을 설정한 후, 수집한 데이터로 가설을 평가하고 추정하는 전통적인 분석이다.
관측된 형태나 효과의 재현성 평가, 유의성 검정, 신뢰구간 추정 등의 통계적 추론을 하는 분석 방법으로 설문조사나 논문에 관한 내용을 입증하는데 사용된다.
두번째는 탐색적 데이터 분석(EDA: Exploratory Data Analysis)로 원 데이터(Raw Data)를 가지고 유연하게 데이터를 탐색하고, 데이터의 특징과 구조로부터 얻은 정보를 바탕으로 통계모형을 만드는 분석방법이다.
주로 빅데이터 분석에 활용하며, 보통 확증적 데이터 분석은 추론통계, 탐색적 데이터 분석은 기술통계로 분류한다.

※ 추론통계 - 수집한 데이터를 이용하여 추론 예측하는 통계 기법으로 신뢰구간 추정, 유의성 검정 기법 등을 이용함
  기술 통계 - 수집한 데이터를 요약 묘사 설명하는 통계 기법으로 데이터의 대푯값, 분포 등을 이용함

탐색적 데이터 분석은 벨 연구소의 수학자 존 튜키가 제안한 데이터 분석 방법으로 통계적 가설 검정 등에 의조한 기존 통계학으로는 새록게 나오는 많은 양의 데이터의 핵심 의미를 파악하는데 어려움이 있다고 생각하여 이를 보완한 탐색적 데이터 분석을 도입하였다고 알려졌다.
데이터를 분석하고 결과를 내는 과정에서 원 데이터에 대한 탐색과 이해를 기본으로 가지는 것이 가장 중요하다.
이에 따라 탐색적 데이터 분석은 데이터의 분포와 값을 다양한 각도에서 관찰하며 데이터가 표현하는 현상을 더 잘 이해할 수 있도록 도와주고
데이터를 다양한 기준에서 살펴보는 과정을 통해 문제 정의 단계에서 미처 발견하지 못한 다양한 패턴을 발견하고 이를 바탕으로 기존의 가설을 수정하거나 새로운 가설을 추가하는 과정을 거친다.
데이터에 대한 관찰과 지식이 이후에 통계적 추론이나 예측 모델 구축 시에도 사용되므로 데이터 분석 단계 중 중요한 단계라고 할 수 있다.

EDA의 목표는 관측된 현상의 원인에 대한 가설을 제시하고, 적절한 통계 도구 및 기법의 선택을 위한 가이드가 되며, 통계 분석의 기초가 될 가정을 평가하고 추가 자료수집을 위한 기반을 제공하는 것이다.

EDA 분석 과정

1. 전체적인 데이터 살펴보기
데이터 항목의 개수, 피처 목록, 결측치, 각 피처가 가지는 데이터 타입 등을 확인하고, 데이터 가공 과정에서 데이터의 오류나 누락이 없는지 확인한다.

2. 이상치(Outlier) 분석
개별 데이터를 관찬하여 전체적인 추세와 특이사항을 관찰한다. 데이터가 많아서 모두 관찰하기 힘들더라도 이상치가 숨어있을 수 있으니, 앞, 뒤, 무작위 추출 기법 등을 활용하여 다각도로 관찰하여야 한다.
이를 조금 더 편하게 하는 방법으로는 요약 통계 지표를 활용하는 방법이 있다. 데이터의 중심을 알기 위해 평균, 중앙값, 최빈값 등을 활용하고,
데이터의 분산도를 알기 위해서 범위, 분산 등을 이용한다.
통계 지표를 이용할 때에는 평균과 중앙값의 차이처럼 데이터의 특성에 주의하여 이용하여야 한다.
마지막으로 시각화를 통해 이상치 분석을 하는 방법이 있다.
시각화 방법으로는 Histogram, Scatterplot, Boxplot, 시계열 차트 등이 있다.
이외에도 기계학습의 K-means 기법, Static based detection, Deviation based method, Distance based Detetction 기법을 활용하여 이상치를 발견할 수 있다.

3. 피처 간의 관계 분석
피처 간의 관계 분석을 통해 서로 의미 있는 상관관계를 갖는 피처의 조합을 찾아낸다. 분석의 대상이 되는 피처의 종류에 따라서 분석 방법 또한 달라져야 한다.
이산형 변수 - 이산형 변수의 경우 상관계수를 통해 두 피처 간의 연관성을 나타낸다.
Heatmap이나 Scatterplot을 이용하여 이를 시각화할 수도 있다.
이산형 변수 - 범주형 변수는 카테고리 별 통계치를 범주형으로 나누어서 관찰할 수 있고, Box plot, PCA plot 등으로 시각화가 가능하다.
범주형 변수 - 범주형 변수의 경우 각 피처 값의 쌍에 해당하는 값의 개수, 분포를 관찰할 수 있고 Piechart, Mosaicplot 등을 활용하여 시각화 할 수 있다.
